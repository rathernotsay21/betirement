# Timeline Images

This directory should contain images for the About page timeline section.

## Required Images

Add the following images to this directory:

1. `timeline-corporate.jpg` - Image representing 28-year corporate career
2. `timeline-bitcoin.jpg` - Image representing Bitcoin discovery in 2017
3. `timeline-retirement.jpg` - Image representing retirement decision in 2024
4. `timeline-today.jpg` - Image representing current lifestyle

## Image Specifications

- Format: JPG or WebP
- Recommended size: 1200x675px (16:9 aspect ratio)
- File size: Optimized to < 200KB each
- Alt text: Descriptive text for accessibility
