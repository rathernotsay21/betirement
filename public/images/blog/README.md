# Blog Images

This directory contains cover images for blog posts.

## Image Requirements

- **Format**: WebP preferred, JPG/PNG acceptable
- **Dimensions**: 1200x630px (optimal for social sharing)
- **File size**: < 200KB for optimal performance
- **Naming**: Use kebab-case matching the blog post slug

## Default Images

- `default.jpg` - Used when no cover image is specified

## Example Images Needed

Based on current blog posts:
- `welcome.jpg` - Welcome post cover
- `bitcoin-basics.jpg` - Bitcoin fundamentals
- `retirement-strategy.jpg` - Retirement planning
- `market-volatility.jpg` - Market analysis

## Author Images

Author avatars should be placed in `/public/images/author/`:
- `michael.jpg` - Main author photo (square, 400x400px)
