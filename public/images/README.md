# Images Directory

This directory contains all images for the Betirement website.

## Directory Structure

- `/timeline/` - Images for the About page timeline section
- `/og-image.jpg` - Open Graph image for social media sharing
- `/twitter-card.jpg` - Twitter card image
- Other images as needed

## Image Guidelines

### Optimization
- Use WebP format when possible for better compression
- Optimize all images to < 200KB for fast loading
- Use Next.js Image component for automatic optimization

### Accessibility
- Always provide descriptive alt text
- Ensure images have proper contrast
- Consider users with visual impairments

### Naming Convention
- Use lowercase with hyphens: `my-image-name.jpg`
- Be descriptive: `timeline-corporate-career.jpg` not `img1.jpg`
- Include size suffix if multiple sizes: `hero-mobile.jpg`, `hero-desktop.jpg`
