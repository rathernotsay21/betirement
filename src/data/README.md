# Data Directory

This directory contains static data files for the Betirement website.

## Structure

- `blog/` - Blog posts in Markdown/MDX format
- `resources/` - Resource metadata and information
- `testimonials/` - Testimonials and success stories

## Blog Posts

Blog posts should be created as `.md` or `.mdx` files with frontmatter:

```markdown
---
title: "Your Blog Post Title"
excerpt: "A brief description of the post"
coverImage: "/images/blog/post-image.jpg"
publishedAt: "2024-01-01"
category: "Bitcoin Fundamentals"
tags: ["bitcoin", "retirement", "investing"]
featured: false
---

Your blog post content here...
```

## Resources

Resources are defined as JSON files with metadata about downloadable content.

## Testimonials

Testimonials are stored as JSON files with customer feedback and success stories.
