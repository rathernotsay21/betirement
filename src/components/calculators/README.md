# Calculators

Interactive calculators for retirement and Bitcoin allocation planning.

## Components

### RetirementCalculator
A comprehensive retirement planning calculator that helps users estimate their retirement savings based on:
- Current age and retirement age
- Current savings
- Monthly contributions
- Expected annual return

### BitcoinAllocationCalculator
Helps users determine the optimal Bitcoin allocation for their portfolio based on:
- Total portfolio value
- Risk tolerance (conservative, moderate, aggressive)
- Investment horizon (short, medium, long-term)

## Usage

```tsx
import RetirementCalculator from '@/src/components/calculators/RetirementCalculator';
import BitcoinAllocationCalculator from '@/src/components/calculators/BitcoinAllocationCalculator';

// In your component
<RetirementCalculator />
<BitcoinAllocationCalculator />
```
